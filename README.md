# Pepperfry Automation Framework

This project is a Selenium-based test automation framework designed to test the Pepperfry website using Java, TestNG, and Maven. It follows the Page Object Model (POM) design pattern for better maintainability and scalability.

---

## 🚀 Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Maven

---

## 📁 Project Structure

src/
├── main/java/pages/ → Page classes (ProductListingPage, ProductPage)
├── main/java/utils/ → Utility classes
├── test/java/base/ → BaseTest setup
├── test/java/tests/ → Test cases (TestRun)

Other Files:
- pom.xml → Dependencies and build configuration
- testng.xml → Test execution configuration

---

## 🧪 Features
- Page Object Model design pattern
- Reusable test components
- Automated product selection flow
- Scroll and dynamic element handling
- Test execution using TestNG

---

## ▶️ How to Run the Project

1. Clone the repository:
   git clone git clone https://github.com/divyA-0427/Pepperfry1.git


2. Open in IntelliJ / Eclipse

3. Run using TestNG:
   - Right click on testng.xml → Run
